HƯỚNG DẪN MỞ FILE CSV TRONG EXCEL VỚI FONT TIMES NEW ROMAN VÀ TIẾNG VIỆT ĐÚNG

===========================================
CÁCH 1: MỞ FILE CSV TRONG EXCEL (KHUYẾN NGHỊ)
===========================================

1. Mở Microsoft Excel
2. File → Open → Browse
3. Chọn file CSV cần mở
4. Trong hộp thoại "Text Import Wizard":
   
   Step 1 of 3:
   - Chọn "Delimited"
   - File origin: Chọn "65001: Unicode (UTF-8)" hoặc "UTF-8"
   - Click Next
   
   Step 2 of 3:
   - Delimiters: Chọn "Comma"
   - Bỏ chọn các delimiter khác
   - Click Next
   
   Step 3 of 3:
   - Chọn "General" cho tất cả columns
   - Click Finish

5. Sau khi file được mở:
   - Chọn toàn bộ sheet (Ctrl+A)
   - Vào Home tab → Font group
   - Chọn font: "Times New Roman"
   - Chọn font size: 11 hoặc 12
   - Click OK

6. Lưu file dưới dạng Excel (.xlsx):
   - File → Save As
   - Chọn định dạng: "Excel Workbook (*.xlsx)"
   - Đặt tên file và lưu

===========================================
CÁCH 2: MỞ BẰNG DATA → FROM TEXT/CSV
===========================================

1. Mở Microsoft Excel
2. Vào tab "Data"
3. Click "From Text/CSV"
4. Chọn file CSV
5. Trong hộp thoại preview:
   - File Origin: Chọn "65001: Unicode (UTF-8)"
   - Delimiter: Chọn "Comma"
   - Click "Load"

6. Áp dụng font Times New Roman:
   - Chọn toàn bộ data (Ctrl+A)
   - Home → Font → Times New Roman
   - Home → Font Size → 11 hoặc 12

===========================================
CÁCH 3: SỬA LỖI ENCODING NẾU ĐÃ MỞ SAI
===========================================

Nếu file đã được mở và tiếng Việt bị lỗi:

1. Đóng file hiện tại
2. Mở lại file CSV bằng Notepad++ hoặc text editor
3. Kiểm tra encoding ở góc dưới bên phải
4. Nếu không phải UTF-8, chuyển sang UTF-8:
   - Encoding → Convert to UTF-8
   - Save file
5. Mở lại trong Excel theo Cách 1 hoặc Cách 2

===========================================
CÁCH 4: TẠO MACRO ĐỂ TỰ ĐỘNG MỞ FILE
===========================================

1. Mở Excel
2. Alt + F11 để mở VBA Editor
3. Insert → Module
4. Paste code sau:

Sub OpenCSVWithUTF8()
    Dim fileName As String
    fileName = Application.GetOpenFilename("CSV Files (*.csv), *.csv")
    If fileName <> "False" Then
        Workbooks.OpenText fileName, _
            DataType:=xlDelimited, _
            TextQualifier:=xlDoubleQuote, _
            ConsecutiveDelimiter:=False, _
            Tab:=False, _
            Semicolon:=False, _
            Comma:=True, _
            Space:=False, _
            Other:=False, _
            FieldInfo:=Array(1, 1), _
            Origin:=65001
    End If
End Sub

5. Chạy macro này mỗi khi cần mở file CSV

===========================================
LƯU Ý QUAN TRỌNG
===========================================

1. File CSV đã được tạo với encoding UTF-8
2. Excel cần được hướng dẫn để nhận diện UTF-8
3. Sau khi mở đúng, áp dụng font Times New Roman
4. Lưu lại dưới dạng .xlsx để giữ nguyên format

===========================================
KIỂM TRA ENCODING
===========================================

Để kiểm tra file CSV có encoding đúng:
1. Mở file bằng Notepad++
2. Xem encoding ở góc dưới bên phải
3. Phải là "UTF-8" hoặc "UTF-8-BOM"
4. Nếu không đúng, chuyển sang UTF-8 và Save

===========================================
HỖ TRỢ
===========================================

Nếu vẫn gặp vấn đề:
- Kiểm tra version Excel (nên dùng Excel 2016 trở lên)
- Cài đặt language pack tiếng Việt cho Windows
- Kiểm tra Regional Settings trong Windows
